# Restaurant Page

A simple restaurant website built with **vanilla JavaScript and Webpack**, where the entire page content is generated dynamically using **DOM manipulation**.

This project was created as part of the **JavaScript curriculum of The Odin Project** to practice modular JavaScript, DOM manipulation, and basic frontend architecture.

---

## Features

- Dynamic page rendering using **JavaScript only**
- Modular code structure using **ES6 modules**
- **Tabbed navigation** (Home, Menu, About)
- DOM elements created entirely with **JavaScript**
- Bundled with **Webpack**
- Styled using **CSS**

---

## Technologies Used

- JavaScript (ES6 Modules)
- Webpack
- HTML5
- CSS3
- Git / GitHub
